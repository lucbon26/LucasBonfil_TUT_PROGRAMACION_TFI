## Descripción

Este Trabajo Integrador Final reúne la resolución de diez ejercicios prácticos orientados al desarrollo de lógica de programación.

A lo largo del trabajo se aplicaron conceptos fundamentales como estructuras condicionales, ciclos iterativos, manejo de cadenas, vectores, matrices, generación aleatoria y cálculos matemáticos.

Para la resolución se utilizaron PSeInt y Python, incorporando además las librerías NetworkX y Matplotlib para la representación gráfica de redes en el ejercicio final.

El objetivo principal fue desarrollar soluciones funcionales, optimizadas y acordes a las consignas propuestas, aplicando criterios propios de análisis e interpretación en cada caso.
